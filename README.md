Highshine Website Project

Project Overview

This project focuses on improving the Highshine website experience through UI/UX enhancements, analytics integration, visitor tracking, and business-focused recommendations. The goal was to create a more engaging user experience, collect actionable visitor insights, and propose data-driven improvements that can increase user engagement and lead generation.

Key Features

1. Modernized landing page design
2. Responsive UI built with React and Tailwind CSS
3. Google Analytics 4 (GA4) integration
4. Custom event tracking (page views and CTA clicks)
5. Visitor analytics dashboard
6. Website restructuring and marketing recommendations
7. Improved animations, hover effects, and visual consistency


Frontend Setup:

cd frontend
npm install
npm run dev


Environment Variables

Create a `.env` file in the frontend directory:

VITE_API_URL=http://localhost:5000
VITE_GA_MEASUREMENT_ID=YOUR_GA4_ID


Backend Setup:


cd backend
npm install
npm run dev

Environment Variables

Create a `.env` file in the backend directory:

PORT=5000
MONGODB_URI=YOUR_MONGODB_CONNECTION_STRING
